# Тема 3. Условия и логические выражения

## Оператор `if`

Условия позволяют выполнять код только в определенной ситуации.

```python
age = 18

if age >= 18:
    print("Доступ разрешен")
```

После условия ставится двоеточие. Код внутри условия пишется с отступом.

## `else`

`else` выполняется, если условие ложно.

```python
age = 16

if age >= 18:
    print("Доступ разрешен")
else:
    print("Доступ запрещен")
```

## `elif`

`elif` используется, когда нужно проверить несколько условий.

```python
score = 75

if score >= 90:
    print("Отлично")
elif score >= 60:
    print("Хорошо")
else:
    print("Нужно повторить")
```

## Операторы сравнения

```python
a == b  # равно
a != b  # не равно
a > b   # больше
a < b   # меньше
a >= b  # больше или равно
a <= b  # меньше или равно
```

Важно: `=` — присваивание, `==` — сравнение.

## Логические операторы

`and` — оба условия должны быть истинными:

```python
age = 20
has_ticket = True

if age >= 18 and has_ticket:
    print("Можно войти")
```

`or` — достаточно одного истинного условия:

```python
is_admin = False
is_moderator = True

if is_admin or is_moderator:
    print("Есть доступ")
```

`not` — отрицание:

```python
is_blocked = False

if not is_blocked:
    print("Пользователь активен")
```

## Частые ошибки

Неправильный отступ:

```python
if age >= 18:
print("OK")  # ошибка
```

Правильно:

```python
if age >= 18:
    print("OK")
```
