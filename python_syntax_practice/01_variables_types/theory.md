# Тема 1. Переменные, типы данных и операции

## Переменная

Переменная — это имя, которое хранит значение.

```python
name = "Anna"
age = 18
height = 1.72
is_student = True
```

В Python не нужно заранее указывать тип переменной. Интерпретатор сам понимает тип по значению.

## Основные типы данных

`int` — целые числа:

```python
count = 10
```

`float` — дробные числа:

```python
price = 99.5
```

`str` — строки:

```python
city = "Moscow"
```

`bool` — логический тип:

```python
is_active = True
```

## Арифметические операции

```python
a = 10
b = 3

print(a + b)  # сложение
print(a - b)  # вычитание
print(a * b)  # умножение
print(a / b)  # деление
print(a // b) # целочисленное деление
print(a % b)  # остаток от деления
print(a ** b) # степень
```

## Преобразование типов

Иногда значение нужно преобразовать из одного типа в другой.

```python
number_text = "25"
number = int(number_text)

price = "19.99"
price_number = float(price)

age = 20
message = str(age)
```

## Проверка типа

```python
x = 10
print(type(x))  # <class 'int'>
```

## Частые ошибки

Нельзя складывать строку и число напрямую:

```python
age = 18
print("Age: " + age)  # ошибка
```

Правильно:

```python
age = 18
print("Age: " + str(age))
```

Имена переменных должны быть понятными:

```python
student_age = 18
```

Лучше не использовать непонятные имена вроде `a1`, `x2`, если смысл переменной важен.
