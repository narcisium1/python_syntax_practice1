# Тема 2. Ввод, вывод и форматирование строк

## Вывод на экран

Для вывода используется функция `print()`.

```python
print("Hello")
print(10)
print("Age:", 18)
```

В `print()` можно передать несколько значений через запятую. Python сам поставит пробелы между ними.

## Ввод с клавиатуры

Для ввода используется функция `input()`.

```python
name = input("Введите имя: ")
print("Привет,", name)
```

Важно: `input()` всегда возвращает строку.

```python
age = input("Введите возраст: ")
print(type(age))  # str
```

Если нужно число, строку надо преобразовать:

```python
age = int(input("Введите возраст: "))
price = float(input("Введите цену: "))
```

## f-строки

f-строки позволяют удобно вставлять значения переменных в текст.

```python
name = "Anna"
age = 18

print(f"Меня зовут {name}, мне {age} лет")
```

Внутри фигурных скобок можно писать выражения:

```python
a = 5
b = 7
print(f"{a} + {b} = {a + b}")
```

## Разделитель и конец строки

У `print()` есть параметры `sep` и `end`.

```python
print("A", "B", "C", sep="-")  # A-B-C
print("Hello", end=" ")
print("world")                 # Hello world
```

## Частые ошибки

Ошибка: забыли преобразовать ввод в число.

```python
a = input("Введите число: ")
b = input("Введите число: ")
print(a + b)  # строки склеятся
```

Правильно:

```python
a = int(input("Введите число: "))
b = int(input("Введите число: "))
print(a + b)
```
